# 云端部署简易指南

## 前言
这份指南将教你如何将应用部署到云端，让任何人都能通过网址访问你的应用！我们选择**Streamlit Cloud**，这是最简单的方式，适合新手操作。

## 准备工作
1. **需要一个GitHub账号**（如果没有，先注册一个）
2. **安装Git软件**（用于上传代码）
3. **创建requirements.txt文件**（告诉云平台需要安装什么软件）

## 第一步：创建requirements.txt文件

1. 在你的`d:\lulu\交易平台`文件夹中，创建一个名为`requirements.txt`的新文件
2. 在文件中写入以下内容：
   ```
   streamlit
   ```
3. 保存文件

## 第二步：安装Git软件

1. 打开浏览器，访问 [Git官网](https://git-scm.com/downloads)
2. 点击下载适合你电脑的版本
3. 安装时使用默认设置即可，一路点击"Next"

## 第三步：创建GitHub仓库

1. 打开 [GitHub网站](https://github.com/)
2. 登录你的账号
3. 点击右上角的"+"号，选择"New repository"
4. 填写仓库名称，比如"my-trading-platform"
5. 选择"Public"（公开）
6. 点击"Create repository"

## 第四步：上传代码到GitHub

1. 在电脑上找到你的`d:\lulu\交易平台`文件夹
2. 在空白处右键点击，选择"Git Bash Here"
3. 在打开的黑色窗口中，依次输入以下命令（每行输入后按回车）：
   
   ```
   git init
   git add .
   git config --global user.email "你的邮箱@example.com"
   git config --global user.name "你的GitHub用户名"
   git commit -m "first commit"
   git branch -M main
   git remote add origin https://github.com/你的GitHub用户名/仓库名称.git
   git push -u origin main
   ```
   
   （注意替换上面的邮箱、用户名和仓库名称）

## 第五步：部署到Streamlit Cloud

1. 打开 [Streamlit Cloud网站](https://share.streamlit.io/)
2. 点击"Sign in with GitHub"登录
3. 在页面上找到并点击创建应用的按钮（可能显示为"New app"、"Create app"或一个加号"+"图标）
4. 选择你的GitHub仓库
5. **分支(branch)填写：`main`** (因为我们在上传代码时使用的是main分支)
6. 主文件路径填写：`main.py`
7. 点击"Deploy!"
8. 等待几分钟，部署完成后会显示一个网址

## 第六步：分享你的应用

部署成功后，你会得到一个类似这样的网址：
`https://yourname-your-app-name.streamlit.app`

你可以把这个网址分享给任何人，他们就可以直接在浏览器中访问你的应用了！

## 常见问题解决

### 问题1：代码上传失败怎么办？
- 检查GitHub用户名和密码是否正确
- 确保仓库名称正确
- 可以尝试重新执行上传命令

### 问题2：部署时出现错误？
- 检查`requirements.txt`文件是否正确
- 确保你的代码在本地能正常运行
- 查看部署日志，根据提示修复问题

### 问题3：如何更新云端的应用？
当你修改了本地代码后，重复第四步的上传命令，云端应用会自动更新！
